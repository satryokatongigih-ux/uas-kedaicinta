Ujian Akhir Semester Web Bisnis - Kedai Cinta

Web app ini dibangun untuk memenuhi kriteria Ujian Akhir Semester (UAS) mata kuliah Web Bisnis. Kedai Cinta merupakan prototype aplikasi pemesanan makanan mandiri (self-service ordering) yang dirancang untuk mempercepat alur transaksi pelanggan.

Kriteria yang Dipenuhi
-  Publikasi online menggunakan Vercel.
-  Perencanaan terstruktur (File PRD.md).
-  Target segmen jelas (Mahasiswa & Solo Diner).
-  Fitur problem solving (Smart Order via WhatsApp & Integrasi Member).
-  Dokumentasi GitHub yang rapi.

Teknologi yang Digunakan
- HTML5
- CSS3 (Custom Styling & Flexbox/Grid)
- Vanilla JavaScript (DOM Manipulation & State Management)
